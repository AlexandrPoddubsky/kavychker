It's normal typography plugin for WordPress
== Description ==

Plugin transform - to &ndash;, make <nobr></nobr> where it needed, make normal quotes.

== Installation ==

1. Upload `kavychker` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
